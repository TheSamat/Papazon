Этап 4. Определение метода, преобразующего входную информацию в желаемое событие

У класса-уведомителя должен быть и метод, генерирующий соответствующие события из входящей информации.

	public void SimulateNewMail(String from, String to, String subject) 
	{
		// Создать объект для хранения информации, которую нужно передать получателям уведомления
		NewMailEventArgs e = new NewMailEventArgs(from, to, subject);

		//Вызвать виртуальный метод, уведомляющий объект о событии
		//Если ни один из производных типов не переопределяет этот метод, объект уведомит всех зарегистрированных получателей уведомления
		OnNewMail(e);
		}
	}
