При определении свойства модификатор доступа с разными уровнями доступа, требуется, чтобы свойство было обьявлено с наименее строгим уровнем доступа, а более строгий был наложен только на один из свойств:

public class SomeType {
  private String m_name;
  public String Name 
  {
    get { return m_name; }
    protected set {m_name = value; }
  }
}
