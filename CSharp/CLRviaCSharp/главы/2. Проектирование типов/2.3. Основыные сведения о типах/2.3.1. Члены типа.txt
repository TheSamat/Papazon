1) Константа - идентификатор, определяющий некую постоянную величину. Константы завязаны на типе, а не на его экземпляре, 
   но на логическом уровне всегда являются статистическими членами. Стоит определять только в том случае, когда есть уверенночть, что
   теоритически нет ни одной возможной причины изменять значение.

2) Поле - значение данных (по сути та же переменная). Поле может быть статичным (частью состояния типа), либо 
   экземплярным (часть состояния конкретного обьекта). 
   Модификаторы:
		static		-	Статичное поле, относящееся к обьекту-типу и инициируемое перед вызовом конструктором типа
		readonly	-	Поле, доступное только для чтения и иницирируется только раз в конструкторе. Конфликтует с volatile
		volatile	-	Поле не будет оптимизировано для многопоточной работы и информирует о возможном изменении поля извне. Конфликтует 
						с volatile. 

Методы: статичные и экземплярные, виртуальные экземплярные и невиртуальные экземплярные, а так же скрывающие базовые  - методы с 
модификатором new

Конструктор экземпляров - метод для инициализации полей экземпляра при его создании. Возвращает ссылку на обьект типа.

Конструктор типа - метод для инициализации статистических полей типа. Выполняется при инициализации (при первом упоминании) обьекта-типа.

Свойства - метод/-ы доступа к полям. Параметризованые свойства и индексаторы.
