Обобщенных делегаты позволяют передавать методам обратного вызова любые типы данных. Так, экземпляры значимого типа
могут передаваться методам обратного вызова без упаковки. Делегат, по сути, - определение класса с 4 методами: 
    конструктор, 
    Invoke, 
    BeginInvoke, 
    EndInvoke. 
А при определении типа делегата с его параметрами, задаются методы класса делегата, т.е. этот делегат:
    public delegate TReturn CallMe<TReturn, TKey, TValue>(TKey key, TValue value);

Компилятор превращает в:

    public sealed class CallMe<TReturn, TKey, TValue> : MulticastDelegate 
    {
        public CallMe(Object object, IntPtr method);
        public virtual TReturn Invoke(TKey key, TValue value);
        public virtual IAsyncResult BeginInvoke(
            TKey key, 
            TValue value, 
            AsyncCallback callback,
            Object object);
        public virtual TReturn EndInvoke(IAsyncResult result);
    }
