Папка со всеми конфигурациями сущностей/обьектов и всем прилагаемым к ним

Конфигурация Product: описание, набор правил и ограничений сущности

Поскольку мы выбрали метод Code First, то необходимо задавать конфигурацию сущностей вручную для каждого типа (или 
использовать конфигурацию по умолчанию). Для этого в EF Core есть 2 способа задания конфигурации:
    1.  Аннотации данных    -   набор атрибутов, накладываемых на поля сущностей
    2.  Fluent API          -   набор методов, определяющихся поставлением между классами и их свойствами, таблицами и их столбцами

Используется Fluent API для оптимизации сопровождения кода из-за легкой читаемости и гикости
