Контекст базы данных, наследующий DbContext и реализующий IPapazonDbContext.

Из-за интерфейса IPapazonDbContext реалезует все свойства.

Для использования же конфигураций сущностей необходимо переопределить метод OnModelCreated, внутри которого необходимо 
использовать метод EntityTypeBilder<T>.ApplyConfiguration, в параметр которого и необходимо передать конфигурацию.

Использует базовый конструктор, передавая в него свойства всех сущностей.
